# Security Policy

The repository contains documentation and an agent instruction package. It does not include application code, hosted services, credentials, or runtime dependencies.

## Reporting Issues

Please open a GitHub issue for:

- misleading guidance that could cause unsafe software design
- unclear trust, permission, or policy recommendations
- packaging problems that could cause an agent to misapply the skill
- documentation defects that materially affect OCS evaluation

If a future version adds executable code, hosted infrastructure, or package publishing automation, this policy should be updated with a private vulnerability reporting path.
